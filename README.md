# Cogniva

Cogniva is an AI-powered learning platform that helps students visualize their learning paths through skill trees and interact with an AI tutor.

## Project Structure

- `frontend/`: React + Vite application for the user interface.
- `backend/`: Node.js + Express backend service with AI integration.

## Getting Started

Refer to the README files in the `frontend/` and `backend/` directories for specific setup instructions.
